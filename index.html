Git Branches Explained

Tạo, chuyển đổi và quản lí các nhánh dự án

git branch : kiểm tra cục bộ nhánh git trên máy tính
git branch -a : kiểm tra nhánh trên github và máy tính dấu * ở phía nhánh nào tức là đang đứng ở nhánh đó
git branch -c tên_nhánh_mới : tạo ra 1 nhánh mới nhưng vẫn ở nhánh hiện tại, chỉ tạo ở máy cục bộ
git checkout tên_nhánh : qua nhánh tên_nhánh
git checkout -b tên_nhánh_mới : tạo ra nhánh mới và qua nhánh đó luôn, chỉ tạo ở máy cục bộ
git push -u origin tên_nhánh : push tên_nhánh lên github
git branch -d tên_nhánh : không đứng ở tên_nhánh, xóa tên_nhánh khỏi máy cục bộ
Nếu tên_nhánh đang thực hiện commit nhưng vẫn muốn xóa tên_nhánh ta thực hiện: git branch -D tên_nhánh
git push -d origin tên_nhánh : xóa tên_nhánh khỏi github
